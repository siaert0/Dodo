# Community Board - Dodo (Team -> personal)
## System

* JAVA
* Spring framework, Spring Security
* Oracle
* Mybatis
* Bootstrap
* AJAX
* jQUERY

## Functions
* 글 읽기, 쓰기, 수정, 삭제(댓글이 있을시 삭제 불가)
* 댓글쓰기, 수정, 삭제 
* 추천하기
* 게시판 구독하기 (개인구독리스트 출력)
* 검색
* Pagenation
* ID/PWD 찾기 
--  PWD 찾기 시 임시비밀번호를 E-Mail로 발송
* File Upload
* 권한설정 - GUEST는 이용사항이 제한됨 
* 비밀번호변경
* 회원탈퇴
